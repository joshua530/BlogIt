A simple blog application for sharing ideas

Features

View posts

Any user is able to any post that have been authored by anyone. A list of the latest posts is displayed on the home page. One can also view posts created by a given user by checking out the user's profile page.

Create Posts

This functionality is for authenticated users only. An authenticated user can be able to create a new post. The user can thereafter edit or even delete the post

Comment on posts

Upon viewing a post, a user may decide to give feedback to the author. This feedback can be given through comments. The comment button is located at the very bottom of a blog post. This functionality is for authenticated users only.

Rest API

The application also offers rest api endpoints. Through these endpoints, the user can be able to perform anything that they could perform through the application's frontend. It is useful for integration with third party systems and even automation of certain actions

About

Inspiration

I have always liked using medium due to it's rich content and ease of use. With inspiration from medium, I decided to build a simple blog application using completely new technologies and within a timeframe of three weeks. The result of that is the BlogIt application. This is my final project for the foundations year at at Holberton school
